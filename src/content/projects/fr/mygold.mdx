Digital gold platform enabling seamless buying, selling, and uploading of real gold with instant wallet syncing.

## Key Features

- Supports leased and non-leased gold options with real-time pricing
- Portfolio tracking with instant wallet synchronization
- Unique gold-leasing model allowing users to earn an additional 5% gold through verified vendor partnerships
- Secure authentication flows and real-time gold pricing
- Push notifications via Firebase Cloud Messaging

## Technical Highlights

- Optimized widget rebuilding and implemented lazy-loading techniques
- Efficient state management for seamless user experience
- Caching strategies for improved performance
- Cross-platform development for Android and iOS
